# src/lib

Pasta para utilitários JS/TS pequenos usados por múltiplas LPs (formatadores, fetchers, helpers de data, etc.).

Mantenha funções curtas e testáveis. Exemplos:

- `src/lib/fetcher.ts` — wrapper fetch/axios.
- `src/lib/format.ts` — formata datas e valores monetários.

Coloque tipos em TypeScript quando apropriado.
